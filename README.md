# qr-gen

C++ QR Code generator using [Nayuki's](https://github.com/nayuki) [qr-code-generator](https://github.com/nayuki/QR-Code-generator) and [tinypngout](https://github.com/nayuki/Nayuki-web-published-code/tree/master/tiny-png-output) c++ code

This is based on the youtube video by [Daniel Hirsch](https://www.youtube.com/@HirschDaniel): [Coding a QR Code Generator in C](https://www.youtube.com/watch?v=duchBA8niWg)
